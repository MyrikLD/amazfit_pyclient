# How to

### Get key:
- Pair band to official app (zepp)
- Use this repo to get key https://github.com/argrento/huami-token.git
- Run main.py

### Example
```bash
main.py -a 00:00:00:00:00:00 -k 79a000a0300001a037a370ee0d219aa9
```
